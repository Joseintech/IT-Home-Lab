# Help Desk Scenarios – Active Directory

## Objective
Demonstrate foundational help desk skills using Active Directory by managing users, organizational units (OUs), groups, and permissions within a simulated enterprise environment.

## Tools & Environment
	•	Windows Server (AWS Cloud)
	•	Active Directory Domain Services
	•	Domain Controller configured with HR, IT, and other organizational OUs

## Tasks Completed
	1	Created organizational units (OUs) to represent departments (HR, IT, etc.).
	2	Added users to department specific OUs.
	3	Created and managed security groups for role based access control.
	4	Assigned users to groups and applied appropriate permissions.
	5	Simulated common help desk tasks:
	6	Resetting user passwords
	7	Unlocking user accounts
	8	Enforcing password policies
	9	Modifying group memberships

## Skills Practiced
	•	Active Directory user and group management
	•	Organizational unit (OU) structuring
	•	Role based access control (RBAC)
	•	Password and account troubleshooting
	•	Permission and access verification
	•	Hands-on help desk support scenarios

## Outcome
Successfully performed core help desk operations in Active Directory, demonstrating practical skills in user management, and troubleshooting within a structured enterprise environment.
