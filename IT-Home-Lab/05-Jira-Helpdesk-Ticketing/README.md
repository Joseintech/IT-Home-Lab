# Jira Help Desk Ticketing Lab

## Objective
Simulate real world IT help desk ticket management using Jira to document and resolve common support requests tied to Active Directory administration.

## Tools Used
- Jira (Kanban board, IT Helpdesk workspace)
- Active Directory (AWS EC2 Domain Controller)

## Tickets Resolved
1. User account lockout - jdoe unlocked via AD
2. Password reset request - jdoe reset via AD Users and Computers
3. New employee onboarding - Jane Smith account created in HR OU
4. Security group removal - John Doe removed from Finance group
5. GPO deployment - Password policy enforced on IT OU

## Skills Demonstrated
- Ticket intake, prioritization, and resolution workflow
- Linking ticketing system actions to AD administration tasks
- Documentation of IT support procedures
- Jira Kanban board management
