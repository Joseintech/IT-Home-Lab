# Environment Setup

## Objective
Design and deploy a local IT home lab on macOS to gain hands-on experience with virtualization, Windows operating system installation, and foundational help desk support tasks.

## Tools & Equipment
	•	MacBook Air (Apple Silicon M4, 16GB RAM)
	•	UTM (virtualization platform for macOS)
	•	Windows 11 ARM64 ISO
	•	Internet connection

## Steps Performed
	1	Created a structured IT Home Lab folder hierarchy to support scalable documentation and future lab projects.
	2	Installed and configured UTM on macOS for Apple Silicon (ARM-based) virtualization.
	3	Downloaded a Windows 11 ARM64 ISO compatible with ARM architecture.
	4	Created and configured a Windows virtual machine, allocating CPU, memory, and storage resources.
	5	Mounted the Windows 11 ISO and completed a full operating system installation.
	6	Verified successful boot, system stability, and access to the Windows desktop environment.

## Skills Practiced
	•	Virtualization fundamentals (ARM-based systems)
	•	Windows 11 installation and configuration
	•	VM resource allocation and system setup
	•	Troubleshooting boot and ISO compatibility issues
	•	Technical documentation and lab organization

## Outcome
Successfully deployed a fully operational Windows 11 virtual machine on macOS using UTM. This environment serves as the foundation for advanced IT lab work, including Active Directory, user management, networking, security fundamentals, and real-world help desk scenarios.
