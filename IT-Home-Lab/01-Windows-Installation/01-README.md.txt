# Windows Operating System Installation

## Objective
Deploy a Windows 11 virtual machine to support hands-on IT help desk and system administration labs.

## Environment
	•	Host OS: macOS (Apple Silicon)
	•	Virtualization Platform: UTM
	•	Guest OS: Windows 11 ARM64

## Implementation
	1	Downloaded and validated a Windows 11 ARM64 ISO.
	2	Created and configured a new virtual machine in UTM.
	3	Allocated system resources (CPU, memory, storage).
	4	Installed Windows 11 through the standard setup process.
	5	Completed initial OS configuration (language, user account, system settings).
	6	Verified successful boot and system stability.

## Skills Demonstrated
	•	Windows operating system installation
	•	Virtual machine provisioning
	•	ARM64 architecture awareness
	•	System configuration and validation
	•	Technical documentation

## Result
Successfully deployed a fully functional Windows 11 virtual machine, ready for Active Directory, networking, help desk troubleshooting, and security-focused lab exercises.
